You what?

A simple layer that allows turn-based multiplayer games to be hosted on the app engine.

Really, what?

Lets developers focus on the logic and UI of the game, leaving the data storage and transport to the framework. We also roll in a bunch of features for free.

Features

  • Lobby
  • User Profiles
  • Multiple Game Management

Sample games

  • Number guessing
  • Naughts and Crosses
  • Conquest
